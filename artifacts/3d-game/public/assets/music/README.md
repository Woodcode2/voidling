# Music drop-in

Place a looping track here as `theme.mp3` and the game plays it automatically
(looped, volume 0.35) instead of the built-in synth score. No code changes
needed — the hook lives in `src/proto3d/audio3d.ts` (`startMusic`).

License requirement for App Store shipping: use CC0 / Pixabay-License /
Kenney-CC0 audio only (no attribution obligations, commercial use allowed).
Keep the source URL + license screenshot in this folder for the record.
