- **Hard size cap:** radius 140. At cap, absorbs give score but no growth.
- **Leader decay** (0.4%/s above the DEVOURER threshold) — verify it actually applies to bots. Log it in debug.
- **Respawn placement:** respawned objects may never appear within 2× radius of ANY void (player or bot). No more feeding stationary giants.

- Retune evolution thresholds for the new curve so a median 3:00 run still peaks at GOBBLER and WORLD EATER stays ~top 10%. Use the debug panel to verify pacing across 3 test rounds.
